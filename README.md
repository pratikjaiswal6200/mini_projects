# 📈 Stock Market Dashboard (Capstone Project)

## 📌 Overview
This project is a React-based Stock Market Dashboard that allows users to search stocks, view real-time prices, and manage a personalized watchlist. It demonstrates core frontend concepts like component-based architecture, API integration, and state management.

---

## 🚀 Features
- 🔍 Search stocks by name or symbol  
- 📊 Real-time stock price updates  
- ⭐ Add/remove stocks from watchlist  
- 🔄 Auto-refresh stock prices  
- 📱 Responsive UI  

---

## 🛠️ Tech Stack
- React.js  
- JavaScript (ES6+)  
- Tailwind CSS  
- Axios  
- Finnhub API
